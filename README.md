# Projekt_AndroidStudio_Rejestracja_na_koncert
Aplikacja Android (Java) pokazująca:
- przekazywanie danych przez Intent
- odbiór wyniku z drugiej aktywności
- walidację formularza
## Funkcjonalność
Użytkownik wprowadza dane, przechodzi do podsumowania i potwierdza lub anuluje rejestrację.
## Technologie
- Java
- Android SDK
